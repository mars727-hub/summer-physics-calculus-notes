# 暑修物理與微積分互動筆記

這是一個為暑修課程整理的靜態互動學習網站，內容包含普通物理與微積分的重點筆記、公式整理與階段複習測驗。

網站連結：

https://mars727-hub.github.io/summer-physics-calculus-notes/

## 內容

### 普通物理

頁面：

https://mars727-hub.github.io/summer-physics-calculus-notes/physics_kinematics_study.html

主題包含：

- 物理測量與 SI 單位
- 量綱分析與單位換算
- 一維運動中的位置、位移、速度與加速度
- 等加速度運動與自由落體
- 位置-時間圖、速度-時間圖的斜率與面積意義
- 微積分在運動學中的基本推導
- 互動式階段複習測驗

### 微積分

頁面：

https://mars727-hub.github.io/summer-physics-calculus-notes/calculus_functions_study.html

主題包含：

- 函數定義、定義域與值域
- 垂直線判別法
- 函數四則運算
- 奇函數與偶函數
- 合成函數
- 指數函數與自然底數 `e`
- 對數函數與對數運算法則
- 互動式階段複習測驗
